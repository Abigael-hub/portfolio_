LLM EVALUATION · OPENAI API · FAILURE MODE ANALYSIS

LLM Output Analyser

Evaluated GPT-3.5-turbo across 30 questions in economics, data analysis, and research methodology — asking not just whether answers were correct but how they failed when they fell short. The pipeline scores each response on concept coverage, accuracy, clarity, and completeness. Key finding: the model scored perfectly on accuracy and clarity but only 0.577 on concept coverage — it answers correctly without always using the precise technical vocabulary the domain requires.
